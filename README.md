Terra & Brasa — Website Responsivo com Bootstrap

Bem-vindo ao Terra & Brasa. Um projeto desenvolvido para prática de HTML, CSS e Bootstrap, simulando uma página moderna para um restaurante especializado em culinária artesanal,
carnes e pratos tradicionais brasileiros.

- Objetivo do Projeto

Este projeto foi criado para praticar:
Estruturação de páginas em HTML5;
Estilização com CSS3;
Componentes e classes utilitárias do Bootstrap 5;
Responsividade (mobile-first);
Organização de arquivos e boas práticas de desenvolvimento front-end;

Tecnologias Utilizadas

HTML5;
CSS3;
Bootstrap 5.3;
Google Fonts;
Font Awesome / Ícones (caso utilizado).
